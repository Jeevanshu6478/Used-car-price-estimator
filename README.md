Machine learning-based system to predict used car prices using regression models and real-world vehicle features.

# 🚗 Car Price Estimation Engine

## 📌 Overview

Developed as a Course Final Project for Supervised and Unsupervised Learning (SUL), this project analyzes and predicts used car prices using machine learning. It focuses on understanding real-world messy data and building regression models to estimate car prices based on multiple features. 

## 📊 Dataset

The project utilizes a custom-generated 1,000-row "dirty" dataset specifically designed to simulate real-world data imperfections. Extensive data cleaning and preprocessing were performed using **Python** and **scikit-learn** prior to analysis and model training to ensure accurate predictions.

## 🛠️ Tools & Technologies
* **Python**
* **scikit-learn**
* **Regression Models**
